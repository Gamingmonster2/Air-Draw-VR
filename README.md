# Air Draw VR | مشروع الرسم الهوائي

تطبيق ويب تفاعلي يعتمد على الذكاء الاصطناعي لتتبع حركة اليد والرسم في الفضاء الافتراضي عبر الكاميرا مباشرة دون الحاجة لأدوات إضافية.

## 🚀 المميزات
- **تتبع دقيق:** استخدام مكتبة MediaPipe Hands لتتبع النقاط المفصلية لليد.
- **واجهة عصرية:** تصميم "Glassmorphism" مع ألوان نيون جذابة.
- **استجابة عالية:** يعمل مباشرة في المتصفح مع دعم كامل لشاشات اللمس والكاميرا.
- **بدون خادم:** المشروع يعمل بالكامل على جانب العميل (Client-side).

## 🛠️ التقنيات المستخدمة
- **HTML5 & CSS3** (Custom Properties & Animations).
- **JavaScript** (ES6+).
- **MediaPipe Hands** (تتبع حركات اليد).
- **Camera Utils** (إدارة تدفق الفيديو).

## 📂 الهيكلية
يتم استضافة المشروع كجزء من سلسلة مشاريع تحت النطاق `me.com.ly` عبر GitHub Pages.
